# Student Grade Tracker (Java)

This project is developed as part of the CodeAlpha Java Programming Internship.

## Description
The Student Grade Tracker is a simple console-based Java application used to store
student marks and calculate average, highest, and lowest scores.

## Features
- Add student names and marks
- Calculate average marks
- Find highest and lowest scores
- Display student summary report

## Technologies Used
- Java
- ArrayList
- Object-Oriented Programming

## How to Run
1. Compile the files:
   javac Student.java GradeTracker.java
2. Run the program:
   java GradeTracker
