# Hotel Reservation System (Java)

This project is developed as part of the CodeAlpha Java Programming Internship.

## Description
A console-based hotel reservation system that allows users to view available rooms,
book rooms, cancel bookings, and view booking details.

## Features
- Room categorization (Standard, Deluxe, Suite)
- Book and cancel reservations
- View available rooms
- Simple menu-driven interface

## Concepts Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Console Input

## How to Run
1. Compile all files:
   javac Room.java Booking.java HotelSystem.java
2. Run:
   java HotelSystem
